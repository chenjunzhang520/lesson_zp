你现在是一个充满创造力的全栈开发专家和用户体验设计师。你的核心任务是根据用户的需求描述，创建具有现代化设计风格、视觉精美、交互性强的应用程序。

项目目标：

创建一个以“立冬快乐”为主题的节日祝福页面。当用户打开页面或点击任意位置时，界面应优雅地弹出多条温暖、诗意或有趣的立冬祝福语（如“愿你暖阳常伴，冬日不寒”、“饺子已煮好，幸福正冒泡”等），每条祝福以轻盈的动画从屏幕中心向外飘散，营造温馨喜悦的节气氛围。

技术要求：

使用 纯 HTML、CSS 和 JavaScript 编写单个自包含文件，无需外部依赖
采用 响应式设计，适配手机、平板和桌面设备
祝福语以 动态弹窗/浮动文字 形式出现，带有淡入、缩放或轻微飘动动画
支持 多次触发（例如点击页面任意处可再次触发祝福雨）
动画流畅自然，避免卡顿，使用 requestAnimationFrame 或 CSS 动画优化性能
设计原则：

整体色调采用冬日暖色系（如米白、浅棕、暖橙、深红），搭配雪花、饺子、围巾等简约插画元素
字体柔和易读，祝福语居中短暂显示后缓缓淡出或飘走
背景可加入极简动态雪花飘落效果（可选但推荐）
用户交互直观——首次加载自动播放一次祝福，之后点击任意位置可重复触发
交付格式：

输出一个完整的 .html 文件，内联所有 CSS 和 JS
代码结构清晰，注释关键逻辑（如动画生成、事件绑定）
包含至少 8 条不同风格的立冬祝福语文本，存储在数组中随机调用
