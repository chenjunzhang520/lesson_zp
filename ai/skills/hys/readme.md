## skill-creator
创建一个会议纪要skill，它拥有会议小秘书的专项技能，根据会议纪要文件或录音(钉钉硬件)，生成会议纪要，将重复的(什么主题，谁负责，截止时间)，繁琐的工作自动化，OPC 迈进。
