# 定位
- 离开文档流？
display: none;
position: absolute;
文档流是html元素默认的布局方式，块级元素垂直排列，行内元素水平排列，遵循从上到下，从左到右的自然顺序。
- 元素在文档流中，离开?
- position: relative;
- 相对定位元素，是相对于元素在文档流中的原来的位置进行定位
- 不会离开文档流，原来在标准流的位子继续占有，
后面的盒子任然以标准的方式对待他。
- display: none;
- 隐藏元素,不占用空间，不会影响
到其他的元素的布局。  
- position :absolute;
  - 需要一个参考标准，这个参考标准是拥有定位元素属性的父元素，如果父容器没有，
  则参考body,离他最近的position 不为static 的元素
  - 会离开文档流

  - position: fixed;
  - 固定定位元素，是相对于浏览器窗口进行定位
  - 会脱离文档流
  - 不会因为因为浏览器滚动而移动

  - position :sticky: 黏连
   根据滚动的位置在relative和fixed 之间切换
   他的行为像relative 定位，直到滚动到指定的阈值
   ，就像fixed定位一样。
   粘性定位的盒子在滚动到指定位置，会固定在指定的位置。

   - position:static;
   - 静态定位元素，是相对于浏览器窗口进行定位
   - 元素按照正常的文档流进行定位
   - 用它来取肖元素的定位属性
   - 不会因为因为浏览器滚动而移动
